What should I do if the app crashes?

If the app crashes, please try the following troubleshooting steps:

  • Make sure your app is updated to the latest version. Check the App Store for updates.
  • Restart your device and try to open the app again.
  • If the issue persists, try reinstalling the app (make sure to back up any data if needed).
  • Clear the app's cache (if the app has this feature).
  • If none of these steps resolve the issue, please contact customer support using the methods below.
